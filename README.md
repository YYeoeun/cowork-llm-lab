# cowork-llm-lab

LLM과 함께 협업(cowork)할 수 있는 Streamlit 기반 작업 도구. 자주 반복되는 사무 작업을 챗봇과 대화하며 처리할 수 있게 만드는 것이 목표이며, 기능은 하나씩 점진적으로 추가해 나간다.

## 목표

- Streamlit UI 위에서 동작하는 LLM 협업 도구
- 작업 흐름은 챗봇 대화 형태로 진행하며, 세부 사항이 모호하면 LLM이 사용자에게 되물어 보완한다
- 결과물은 화면에 미리 보여주고, 파일로도 내려받을 수 있도록 한다

## 기능

### 1. 엑셀 통합 (첫 번째 기능)

여러 개의 엑셀 파일을 업로드해 한 파일로 통합한다.

- 업로드한 각 파일의 **모든 시트 내용**을 읽어 저장
- 다른 항목은 동일하고 **수치만 다른** 행/셀이 발견되면, 어떻게 합칠지(합산, 평균, 최신값 사용, 사용자 지정 규칙 등) 사용자에게 묻는다
- 사용자가 자연어로 통합 규칙을 지정하면 그대로 적용 (예: "B열은 합계로, C열은 평균으로 묶어줘")
- 결과 엑셀을 화면에 표시하고 `.xlsx` 파일로 다운로드 제공

### 향후 추가 예정

추가 기능은 사용자가 필요로 하는 흐름을 따라 하나씩 확장해 나간다.

## 기술 스택

- **UI**: Streamlit
- **LLM**: 선택적 백엔드 (OpenAI / Gemini / Ollama 등 — 추후 정리)
- **데이터 처리**: pandas, openpyxl

## 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

> 현재 초기 설계 단계로, 구현은 기능 단위로 순차 진행 중이다.
